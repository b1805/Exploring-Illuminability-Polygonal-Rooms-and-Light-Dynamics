Exploring Illuminability: Polygonal Rooms and Light Dynamics

I started off the summer with creating a program called Custom Star Room Illumination (Star Customizer). Custom Star Room Illumination is an interactive web-based application that simulates light rays bouncing off walls in a customizable star room. You can adjust the number of light rays, the number of triangles forming the room, precision, photon thickness colors, and animation speed. You can also switch to the angle version to shoot the rays one at a time at your chosen angles and colours. Additionally, you can record the simulation and download the video. This also counts the number of active photons. The theory behind this is that any star room made with an even number of triangles >=6 (6,8,10,12....), when rays of light is shot from its midpoint in all directions, would never return back to the midpoint but illuminate the rest of the room completely. It is as if you're standing in the middle of the room with a lamp, an observer can see everything in the room except for you!

Next, I made the Double Star program. Double star program is similar to the Star_customizer with all the features including adjusting the number of light rays, precision, photon thickness colors, animation speed, recording, the option to switch to the angle version to shoot the rays one at a time at your chosen angles and colours and the active photon counter. This program lets you use all these features for Double Star created using 20 isosceles triangles ABC with A=30=B, C=120. This figure has 2 unilluminable points. When rays of light are shot from one of these points in all directions, it would never return back either of them but illuminate the rest of the room completely. It is as if you're standing at one of these points with a lamp, an observer can see everything in the room except for you and the other point!

Next, I made the Custom Un-illuminable Room Finder (Triangle Clicking) program. This program is another interactive web-based application that lets you create your own new unilluminable rooms. You can simply select the right-angle isosceles triangles you like and then click create shape. This has a wide range of options to customize, including the location and zoom of upto three magnification boxes for the Un-illuminable point, number of light rays, speed, precision, colours, size of the triangle grid, partition and post, photon thickness, location of the light source, option to save and load the created shape and selected settings, etc. You can also switch to the angle version to shoot the rays one at a time at your chosen angles and colours. Additionally, you can record the simulation and download the video and you can remove or keep the grid lines while recording. This also counts the number of active photons. Couple of notes: Firstly, select triangles using left mouse click, right click changes the location (coordinates) of the light source and the (small) magnification box. You can individually change their coordinates using the controls on the right. You can move the big magnification box by dragging it! Secondly, if you're loading both Settings and Shape, load Settings first before loading the Shape.

Next, I made the Custom Room Illuminator (Snakes & Bushes) program. This program is another interactive web-based application that lets you find points to illuminate any room. First select all the customizable settings including the grid size, precision, photon thickness, distance from the source, partition and post, speed, M & N, light source coordinates, magnification and colours. Then select the triangles you like or switch to a square grid and select the squares you like. If you want your light source coordinates to be autopicked, hit the first create shape button otherwise the second one which will apply your chosen light source coordinates. Now you can pick from different animation options and 3 modes of animation, play around with these features and use the dragging (by holding and moving the mouse) and zoom features to find points to fill all polygonal rooms! Additionally, you can record the simulation and download the video and you can remove or keep the grid lines while recording.

Lastly, I worked on the Beams Splitting program which simulates the splitting of a beam starting at any point in a right-angled isosceles triangle or a square. You can pick the size of the shape, the angle span of the beam, precision, speed of propagation of the beam, starting point of the beam, colours and magnification! You can record these finding for further investigation!
